Midpoint

About Us

Midpoint was created by Alex West, Hannah Whitmore, Kamala Hasanova, Febby Chang, and Hana Kastrati. The Girls Who Code 2017 Summer Immersion Program at Goldman Sachs gave us the skill set necessary and opportunity to build the site. Throughout the weeks we spent learning alongside one another we were able to become close and see our common interests. For some of us, Girls Who Code was a first experience in computer science, making it all the more exciting to develop a program that could directly help people in their daily lives.
